# Sales Performance & Growth Analytics Dashboard

## Project Overview
This dashboard is designed to analyze sales and profit performance using Power BI. It helps identify trends, product performance, regional profitability, and overall business growth insights.

## Features
- Total Sales, Total Profit, and Average Monthly Sales KPIs
- Sales trend analysis by month
- Sales by sub-category
- Profit by region (donut chart)
- Quantity by segment (treemap)
- Sales growth vs target (gauge chart)
- Profit breakdown (waterfall chart)
- Interactive slicers (Year, Category)

## Tools Used
- Microsoft Power BI
- Sample Superstore Dataset (Kaggle)

## Files in Repository
- `Dataset` – sample_-_superstore.csv
- `PowerBI_Dashboard.pbix` – Power BI source file
- `screenshot` – ScreenShot of dashboard
- `README.md` – Project documentation

## How to Use
1. Download the `.pbix` file.
2. Open in Power BI Desktop.
3. Interact with slicers and visuals to explore insights.
